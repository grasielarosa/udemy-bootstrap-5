Cores aplicadas a textos

O bootstrap trabalha com cores baseadas em propósitos. A cor primária que é a principal cor do meu site, a secundária que é a segunda mais importante, operação realizada com sucesso, ou erro, ou informativa. Em resumo, definir as cores com base no que queremos informar.

class text-info

class text-black

class text-white com bg-dark

class text-black-50 (como 50% de opacidade - bom para usar sobre imagens)

class text-white-50 com bg-primary (com 50% de opacidade)
